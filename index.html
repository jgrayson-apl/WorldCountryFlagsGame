The World Country Flags Game

Drag a flag onto the map and drop it over the corresponding country. Your score will change based on the points assigned to each flag. The game will finish after 10 incorrect guesses or when there are no more flags available. Note: this game is designed for use with newer versions of these browsers: Chrome, Firefox, Edge, and Safari.
This application uses information from the following sources:
  • Wikipedia - Gallery of sovereign state flags
  • Esri Living Atlas - World Countries Feature Layer
  • Esri Living Atlas - Charted Territory Basemap
